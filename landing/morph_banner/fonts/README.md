# Fuentes para el morph banner

Colocá aquí los archivos `.otf` / `.ttf` que quieras usar con el pipeline (`landing/scripts/font_pipeline.py`).

**Procedimiento completo:** [docs/MORPH_BANNER_FONT.md](../../../docs/MORPH_BANNER_FONT.md) (ejecutá los comandos desde el directorio `landing/` donde está `manage.py`).

**Nota:** Si no movés la fuente aquí, pasá la ruta absoluta o relativa con `--font` al ejecutar el pipeline.

La fuente debe incluir glifos Unicode para `A-Z`, `a-z` y `0-9` en el cmap.
