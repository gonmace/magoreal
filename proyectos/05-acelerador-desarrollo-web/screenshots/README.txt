Mockups generados para el Proyecto 5 (Acelerador de desarrollo web a medida).

Los 4 PNG fueron creados programáticamente con PIL porque este proyecto es un stack / backbone técnico, no tiene una UI propia que mostrar. Los mockups ilustran el valor del producto de forma visual.

Archivos:

01-stack-completo.png  (1400x880)
  Diagrama del stack con 9 componentes (UI, Core, Workflow, AI Integration, DB, Cache,
  Proxy, Orchestration, Security) + sección de beneficios para el negocio + flujo de
  deploy en 4 pasos.
  Útil para la sección "Qué incluye el acelerador".

02-deploy-flow.png  (1280x820)
  Mockup tipo terminal mostrando un deploy real de cero a producción:
  - git clone
  - make setup (wizard con dominio, n8n, MCP, etc)
  - make nginx + certbot
  - make deploy con todos los pasos
  - Cierre: "App en producción en 4 min 12 s"
  Útil para la sección "El deploy real, en tus ojos".

03-mcp-integracion.png  (1400x820)
  Diagrama de la integración Claude Code / Cursor ↔ n8n-MCP ↔ n8n ↔ Django con
  ejemplos concretos de prompts y tools expuestas. Remata con 4 razones de por qué
  esto cambia el juego.
  Útil para la sección "Tu app conectada con IA desde el día 1".

04-timeline-comparacion.png  (1400x820)
  Comparación lado a lado: desarrollo tradicional (60 días al primer valor) vs. con
  este acelerador (3 horas al primer deploy, 20 días al MVP completo).
  Útil para la sección "Días, no meses".
